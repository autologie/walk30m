縮退運転について（2017.7.15）

「30分でどこまでいける？」は現在縮退運転中です。

このページではユーザーの皆様への縮退運転の影響や、縮退運転に至った経緯を説明いたします。

縮退運転の影響

縮退運転中は時間帯によってサービスがご利用いただけないことがあります。
開始地点をどこに設定しても計算が進まない場合は、お手数ですが時間帯を変えて再度お試しください。

日頃からご利用いているユーザーの皆様にはご不便をおかけして申し訳ありません。

ご協力ありがとうございます。

なぜ縮退運転なのか？

利用者数の増加を背景として、毎月のサービスの運用費用を支出することが難しくなってきたためです。

当初から、本サービスは余暇時間を利用したサイドプロジェクトとして、 サイトコンテンツの配信から計算処理のための API 利用、ログ収集に至るまで、 概ねパブリッククラウド（Google Cloud Platform, Amazon Web Services）が提供する無料利用枠の範囲内で運用してきました。

運用開始から3年近く経過しましたが、この間に少しずつではあるものの、サイトへのトラフィックおよびサービスの利用（xx分でいける範囲の計算）が増加してきました。

そして予期されていたことですが、これに伴って数ヶ月前から一部のクラウドサービスの使用について無視できない請求が発生するようになりました😱

ここに至って広告の掲載やサービスの一部または全部の有料化の検討を開始しましたが、 いずれもただちに実現可能ではなかったため、当面の措置として縮退運転に踏み切ることにいたしました。

縮退運転の期間

運用費用回収の目処が立つまで無期限に縮退運転とする予定です。
